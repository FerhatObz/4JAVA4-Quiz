# 🚀 4JAVA4 Quiz

**4JAVA4 Quiz**, kendi İngilizce kelime ve cümlelerinizi oluşturup düzenleyebileceğiniz modern bir Android uygulamasıdır.

Kelimelerinizi kategorilere ayırabilir, Flashcard (Kart Çevirme) sistemiyle çalışabilir, Quiz çözebilir ve ilerlemenizi takip edebilirsiniz.

Bu proje tamamen öğrenme ve geliştirme amacıyla açık kaynak olarak paylaşılmıştır ve aktif olarak geliştirilmektedir.

---

# ✨ Özellikler

- 📂 Sınırsız kategori oluşturma
- 📖 İngilizce - Türkçe kelime ekleme
- 🃏 Flashcard (Kart Çevirme) sistemi
- 📝 Quiz modu
- ⭐ Favorilere ekleme
- ✅ Öğrenildi olarak işaretleme
- 💬 Sık kullanılan cümleler
- 🔍 Anlık arama
- 🌙 Premium Dark Tema
- 💾 JSON ile yedekleme ve geri yükleme

---

# 📱 APK

Android Studio kurmadan uygulamayı denemek isterseniz APK dosyası aşağıdaki dizinde bulunmaktadır.

```text
app/build/outputs/apk/debug/app-debug.apk
```

İsterseniz projeyi Android Studio ile açıp kendi APK'nızı da oluşturabilirsiniz.

```
Build → Build APK(s)
```
